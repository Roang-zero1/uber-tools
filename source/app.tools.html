app.tools.runhelper.runcmd(*popenargs, **kwargs)[source]¶

Run command with arguments and return the std_out, std,err and returncode as a namedtuple.

If the exit code was non-zero it raises a CalledProcessError. The CalledProcessError object will have the return code in the returncode attribute and output in the output attribute.

The arguments are the same as for the Popen constructor. The stdout and stderr argument is not allowed as they are used internally.

Parameters:
  • *args – Variable length argument list.
  • **kwargs – Arbitrary keyword arguments.
Returns:

The output created by the executed command.
Return type:

CmdOutput

Example

>>> runcmd(["ls", "-l", "/dev/null"])
